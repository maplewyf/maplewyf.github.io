HTTP响应状态代码

Hexo
2020-02-08
  • 信息响应（100-199）
    • 100 Continue 表示迄今为止所有的内容都是可行的
  • 成功响应
    • 200 OK 请求成功
  • 重定向
    • 301 请求永久重定向的状态码
    • 300 Multiple Choice 被请求的资源有一系列可供选择的回馈信息
    • 302 Found 请求的资源现在临时从不同的URI响应请求
  • 客户端响应
    • 400 Bad Request 语义有误；请求参数有误
    • 403 Forbidden 服务器已经理解请求，但是拒绝执行它
    • 404 Not Found 请求失败，请求所希望的资源未在服务器上被发现
    • 408 Request Timeout 请求超时。客户端没有在服务器预备等待的时间内完成一个请求的发送。
  • 服务器端响应
    • 500 Internal Server Error 服务器遇到了不知该如何处理的情况
